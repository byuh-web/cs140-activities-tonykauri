3. what is the difference between a SPAN tag and a DIV tag?

A span tag is an inline element used for small content, while a div tag is a block-level element used to group larger sections.

4. What is the difference between a DIV tag and a HEADER, MAIN>, or FOOTER?

A div tag has no semantic meaning, while header, main, and footer are semantic elements that describe the role of the content.

5. What are the different ways to select a color?

Colors can be selected using color names, HEX values, RGB, RGBA, HSL, and HSLA.
